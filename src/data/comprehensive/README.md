# Comprehensive Content Library for National Psychology Examination

This directory contains the most comprehensive content library for the Australian National Psychology Examination study app. The content is designed to be 40% harder than the actual exam level, covering all edge cases and complex scenarios that provisional psychologists might encounter.

## Content Standards

### Evidence-Based Requirements
All content must be evidence-based and reference:
- APS Code of Ethics (with section numbers)
- Relevant legislation (with specific acts/sections)
- DSM-5-TR criteria (with page numbers)
- Current best practice guidelines
- Recent research (2020-2024)

### Language Requirements
- Australian spelling throughout
- Local terminology (e.g., "mental health plan" not "treatment plan")
- State/territory specific requirements noted
- Indigenous cultural protocols included

### Difficulty Calibration
- 40% harder than exam level
- 40% at exam level
- 20% foundational (still challenging)

### Content Quality Standards
Each content piece must include:
- Clear learning objectives
- Practical application
- Common pitfalls
- Exam strategy tips
- Real-world relevance
